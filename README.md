# LoL-Draft-Optimizer
LoL Draft Optimizer est un outil d'analyse de données conçu pour optimiser la phase de sélection des champions (Draft) dans League of Legends. En utilisant les données réelles issues de l'API Riot Games, l'outil calcule les synergies entre alliés et les contres (counter-picks) face aux adversaires pour maximiser les chances de victoire.
